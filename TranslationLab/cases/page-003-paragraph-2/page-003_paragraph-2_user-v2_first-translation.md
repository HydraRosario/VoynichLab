## Hallazgo principal

La V2 cambia el campo semantico del parrafo.

Antes el analisis giraba alrededor de `hete`, `hate`, `hora`.

Ahora aparece una raiz mucho mas fuerte y repetida:

- `crime`: crimen, falta grave, pecado, iniquidad, asesinato en Asturian.

Esto hace que el parrafo parezca menos botanico/descriptivo y mas normativo, moral, medico-ritual o juridico-simbolico.

## Traduccion tentativa V2

Lectura literal, sin embellecer:

> Conector-ciclo-crimen. Algo de familia `teo`. `laol-met-otra`. Ora/ciclo-crimen. Estado/calor/nombre-crimen. `laol-meto` con familia mir/nir y cierre. `heteo` ciclo y forma de crimen. Reaparece ciclo con cierre `ra`. Aparece una zona `hate/ate` ligada a odio/ira/higado. Luego vuelven formas de ciclo, familia mir/nir, y cierres `ra/ria/rine`.

Lectura semanticamente mas compacta:

> El parrafo parece organizar una serie de formulas sobre ciclo/tiempo y transgresion/falta. `crime` aparece como nucleo fuerte, mientras `hora/ora` parece marcar el ciclo o fase. Hacia el medio aparece `hate/ate`, que puede introducir odio, ira, dano, oposicion o una lectura anatomica de higado. El cierre recurrente `ra/ria/rine` parece flexionar o cerrar las unidades.

Traduccion candidata muy tentativa:

> "En el ciclo de la falta/crimen; se nombra o se calienta la falta; una raiz `laol/met` modifica otra fase; la falta vuelve en el ciclo; aparece odio/ira o una marca corporal; las fases se cierran con `ra/ria/rine`."

## Lo que si podemos decir

1. `crime` ahora es la raiz semantica mas fuerte del parrafo.
2. `hora/ora` es la raiz estructural mas repetida.
3. `hate/ate` probablemente no debe descartarse: puede estar emparentada por elision de `h`.
4. `teo/teoa/toa/ateo/eteo/heteo` es una familia critica. Si resolvemos esa familia, el parrafo avanza mucho.
5. `ra/ria/rine` parece cierre o flexion, no raiz principal.

## Siguiente prueba

La proxima busqueda no debe ser por raices sueltas solamente.

Hay que buscar familias:

- `crime / crima / crine / cri`
- `hate / ate / ateo / atera`
- `teo / teoa / teora / toa / eteo / heteo`
- `hora / ora`
- `ra / ria / rine`

Si esas familias producen un campo semantico estable, tenemos el primer esqueleto real de traduccion.
