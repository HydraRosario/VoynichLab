# Research Writing Workspace
This directory organizes possible future publications. It is not the canonical
evidence store and it does not define experiment outcomes.

| File | Purpose |
|---|---|
| `CLAIMS.md` | Statements currently supported, interpreted, or still open |
| `OUTLINE.md` | Working publication structure and research roadmap |
| `FIGURES.md` | Figure plan linked to evidence |
| `LITERATURE.md` | Related-work map |
| `NOVELTY-MATRIX.md` | Working comparison against prior literature |

Evidence remains canonical in frozen releases, public artifact bundles, and the
research registry. Writing may evolve without rewriting published evidence.
