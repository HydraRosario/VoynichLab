# VoynichLab Falsification Protocol

This project must not assume the shorthand hypothesis is true.

The scientific task is to test whether the method produces structured, predictive, reusable evidence that is hard to explain as post-hoc interpretation.

## Core Hypothesis

`visible glyph -> shorthand superstructure -> visual parts/strokes -> romanized reading -> etymological roots -> semantic translation`

## What Would Support The Hypothesis

1. The same visual stroke family maps to the same base value across pages.
2. Small curve-ending differences consistently predict different endings.
3. Contextual rules, such as `h -> t`, predict unseen examples.
4. A second paragraph can be annotated using rules inferred from the first paragraph.
5. A model or script can suggest readings above chance before manual correction.
6. Root segmentation produces repeated semantic patterns, not arbitrary one-off explanations.

## What Would Weaken Or Break The Hypothesis

1. The same visual form receives unrelated readings without explicit rule.
2. Rules are invented only after seeing each example.
3. Color/value mappings collapse when tested on a second paragraph.
4. Entropy or Zipf results are similar to naive controls.
5. Root searches accept too many languages/meanings and become unfalsifiable.
6. Translation cannot predict anything new.

## Statistical Warnings

The first packet has 23 units. That is enough for an evidence prototype, not enough for proof.

Do not claim:

- Zipf proven
- entropy proves language
- fraud disproven
- translation validated

Allowed claim:

- A complete pilot paragraph has been structurally annotated and can now be tested for consistency.

## Current Scientific Status

Page 003 / paragraph 2 is a complete pilot packet:

- 23 units
- 86 visual parts
- 199 strokes
- 0 missing readings

Phase 2 is not "prove the theory". Phase 2 is:

- extract rules
- test internal consistency
- define controls
- prepare a second paragraph as a held-out test

## Decision Rule

Before annotating large volume:

1. Freeze the rule legend from paragraph 2.
2. Annotate a second paragraph using the frozen legend.
3. Count contradictions.
4. Only update the rule system if contradictions are explainable and reusable.

If the rule legend changes freely for every paragraph, the method is not scientific.
