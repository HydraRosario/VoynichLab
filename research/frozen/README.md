# Repository-Level Frozen Evidence
This is the canonical location for new repository-level scientific freezes.

Current release:

- `CORPUS-V2-AUDITED/` — the six-folio audited corpus and its replay record.

Historical V1 and GRAMMAR-V1 material predates this convention and remains in
its published lab-local path. See `research/FROZEN-EVIDENCE.md` for the complete
catalog.

directories are append-only: supersede them with a new version; never silently
rewrite them.
